# Exercícios de Física e Lógica em Python

Repositório com exercícios desenvolvidos para praticar lógica de programação e cálculos físicos utilizando Python.

## Exercícios

### 1. Cálculo de Força
Programa que calcula a força necessária para desacelerar um carro utilizando massa e desaceleração.

### 2. Energia Cinética
Programa que calcula a energia cinética de uma bola de boliche com base na massa e velocidade.

### 3. Distância de Frenagem
Programa que calcula a distância percorrida por um trem durante a frenagem.

### 4. Cálculo de Instâncias e Custos
Programa que calcula a quantidade de instâncias necessárias e o custo total baseado na quantidade de vídeos.

### 5. Conversão de Distância
Programa que converte uma distância em milímetros para quilômetros, metros, centímetros e milímetros restantes.

## Assuntos praticados

- Variáveis
- Entrada e saída de dados
- Operações matemáticas
- Fórmulas físicas
- Divisão inteira e resto
- Formatação de números
- Lógica de programação
